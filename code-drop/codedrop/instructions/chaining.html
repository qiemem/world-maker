Chaining

You can also do multiple things in the same statement.
Just like saying hand makes it so that you're talking to the hand, methods also let you talk to something.
For forward, it lets you talk to the hand again. Try entering hand.forward(1.0).cube();. This tells the hand to move forward and then create a cube!
For cube, it lets you talk to the newly created cube. Try entering hand.cube().forward(1.0);. This tells the hand to create a cube, and then tells the cube to move forward!
